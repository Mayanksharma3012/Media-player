# HTML5 Media Player

A simple media player built using HTML5 and CSS.

## Description
This project demonstrates the use of HTML5 media elements with clean layout and styling using CSS. Native media controls are used for playback.

## Features
- HTML5 video/audio support
- Responsive layout
- Clean and simple UI
- CSS-based styling

## Tech Stack
- HTML5
- CSS3
